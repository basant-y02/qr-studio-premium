# QR Studio Premium

![QR Studio Preview](https://github.com/basant-y02/qr-studio-premium/blob/main/Screenshot%202026-03-15%20145542.png?text=QR+Studio+Premium+Preview)

**QR Studio Premium** is an all‑in‑one web tool that lets you design beautiful, fully customized QR codes. Control every detail: colors, dot shapes, frame styles, add your own logo, and include a caption. All with a sleek, glassmorphism interface that works on desktop, tablet, and mobile.

---

## ✨ Features

- **Full Customization** – Change QR code color, background color, frame color, dot shape (square, dots, rounded, classy), and frame style (none, simple, double, rounded, neon).
- **Logo Upload** – Add a center logo (PNG, JPEG, GIF, WebP) with adjustable size and background shape (square, circle, no background).
- **Text Under QR** – Add a caption like "SCAN ME" – its color automatically matches the frame color.
- **Smart Templates** – Quickly generate QR codes for:
  - **Instagram** (pre‑filled URL with pink/purple theme)
  - **WiFi** (enter SSID, password, encryption)
  - **Website** (simple URL template)
  - **Contact** (vCard with name, phone, email, website)
- **High‑Quality Downloads** – Choose download resolution (Low, Medium, High) and export as **PNG**, **JPG**, or **SVG** (vector).
- **Autosave** – All settings are saved in your browser’s local storage – come back later and your design is still there.
- **Drag & Drop Logo** – Drag an image directly onto the preview area to set it as the logo.
- **Copy to Clipboard** – One‑click copy of the current text/URL.
- **Fully Responsive** – Works perfectly on phones, tablets, and desktops.

## 📦 How to Use

1. **Enter your text or URL** in the input field.
2. **Customize the look** using the controls on the left:
   - Pick QR and background colors.
   - Select a frame style.
   - Change the dot shape.
   - Upload a logo (optional).
   - Adjust logo size.
   - Add a caption.
3. **Use a template** for quick setup: Instagram, WiFi, Contact, or Website.
4. **Download your QR code** as PNG, JPG, or SVG. Choose the resolution you need.

All settings are automatically saved – you can close the page and come back later to continue where you left off.

## 🎨 Customization Options

| Control            | Description                                                                 |
|--------------------|-----------------------------------------------------------------------------|
| **URL / Text**     | The data to encode (URL, plain text, WiFi config, vCard, etc.)             |
| **Logo**           | Upload an image to appear in the center. Supports PNG, JPEG, GIF, WebP.    |
| **Frame Style**    | None, Simple, Double, Rounded, Neon – each with your chosen frame color.   |
| **QR Dots Style**  | Square, Dots, Rounded, Classy – changes the shape of the QR modules.       |
| **Logo Shape**     | Square (with margin), Circle, or No Background.                             |
| **Colors**         | QR color, Background color, Frame color (also used for caption bar).       |
| **Logo Size**      | Slider to control the relative size of the logo (0.10 – 0.40).              |
| **Text Under QR**  | Optional caption displayed below the QR code.                               |
| **QR Size**        | Adjust the pixel dimensions (150px – 400px).                                |
| **Download Resolution** | Low (1x), Medium (2x), High (4x) – affects file size and quality.     |

## 📸 Templates

- **Instagram** – Sets a typical Instagram URL, pink colors, and the Instagram logo.
- **WiFi** – Shows extra fields for SSID, password, and encryption. Generates a WiFi‑connect QR.
- **Website** – A clean template for any website URL.
- **Contact** – Provides fields for name, phone, email, website, and creates a vCard QR.

---

## 📥 Download Formats

- **PNG** – Best for web and general use (supports transparency).
- **JPG** – Smaller file size, no transparency.
- **SVG** – Vector format, infinitely scalable.
